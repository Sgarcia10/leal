## Description

Angular 6 Project

## Installation

```bash
$ npm install
```

## Running the client

```bash
# Development (watch mode)
$ npm run client:dev

# Production build
$ npm run client:build
```

## Test

```bash
# unit tests
$ npm run test or $ npm t

# unit tests watch
$ npm run test:watch

# test coverage
$ npm run test:cov
```
